Developer Instructions

Assuming you're comfortable with a terminal shell this will get you started:

# Somewhere to work...
mkdir ~/website && cd ~/website

# ...and a virtualenv
python -m venv staticsite_env
staticsite_env/bin/activate

# Install Django & Staticsite
pip install Django django-staticsite

# Begin project
startstaticsite my_site

# Run development server to preview
cd my_site
python manage.py runserver

# View in Browser
open http://127.0.0.1:8000

# Make your edits here

# When you're happy...
python manage.py buildstaticsite
python manage.py collectstatic

# See your site, ready for rsync...
tree build
